遺伝子組み換え食品、ゲノム編集食品とは何か？

遺伝子操作食品とは遺伝子を人為的に組み換えたり、それを生細胞に移入したりして操作した食品です(1)。遺伝子操作には遺伝子組み換え食品、ゲノム編集食品が存在します。遺伝子組み換え食品とはバイオテクノロジー技術のなかの遺伝子組換え技術を応用した食品で、GM食品ともいわれています(2)。遺伝子組み換え作物は「他の生物から取り出した遺伝子をゲノムに組み込むことで作られます。その結果、その作物は新しい性質を持つようになります。」

2015年（平成27）11月12日の時点で、日本国内で製造・輸入・流通・販売されている遺伝子組換え食品はダイズ、ナタネ、ジャガイモ、トウモロコシ、ワタ、テンサイ、アルファルファ、パパイアの8作物（303品種）と、酵素などの21品目の食品添加物が認められている。


その一方、ゲノム編集食品とはゲノム編集技術を用いて品種改良された農作物や家畜・魚などから得られる食品(3)、DNAを切断する人工酵素を使ってDNAに突然変異を起こす技術を使った食品である。例にジャガイモが挙げられる。ジャガイモは本来芽に毒素があるが、ゲノム編集を使用して芽に毒素を作らないジャガイモを作ることができる。ジャガイモ以外にも血圧降下作用が期待されるGABAを多く含むトマトや筋肉量を増やしたタイなどが作られている。(4)

3．ゲノム編集食品のメリット、デメリット

1)メリット

　ゲノム編集食品のメリットは2つあります。 ・短期間で品種改良が可能である ・安全性審査を受ける必要がない 新たに遺伝子を入れる遺伝子組み換え食品は、それによって人間に害を及ぼすことがないか、国の安全性審査を受けることが義務づけられています。 （企業は成分解析や動物実験などを行い、アレルギーの原因物質や発がん性物質などが新たに生み出されていないことを確認します。そのデータを国の食品安全委員会に提出。厳格な審査を受けた上で、許可を得る必要がある) ですが、ゲノム編集食品はもともとある遺伝子を切るだけとして、日本は安全性審査は必要ではないと考えられています。(6) ですから、安全性審査を必要としないゲノム編集食品は多くの食品を早く国民に食べてもらうことができ、ゲノム編集食品の発展も早くできると考えられます。


2)デメリット

ゲノム編集食品のデメリットは2つあります。 ・未知のリスクがある ゲノム編集食品には「オフターゲット」という狙った遺伝子ではない別の遺伝子を切ってしまうことなどをはじめとする「未知のリスク」があります。(8)

・安全性の認知度の低さ 大半の研究者の間では「新しい作物を生み出す技術的な観点や安全性のうえで、オフターゲット問題は大きな問題ではない」(9)との意見が圧倒的に多いといいます。 しかし、東京大の研究チームが一般市民を対象に意識調査をしたところ、ゲノム編集食品を「食べたくない」と答えた人が4割を超えた(10)といいます。この結果からゲノム編集食品の安全性を国民に伝え、流通させる必要があります。


3)問題点

私たちは遺伝子組み換え食品のデメリット、健康被害が出ているという問題は遺伝子組み換えという仕組み自体が不完全なものであるため、安全性審査が不要である安全性が確保されたゲノム編集食品のデメリットを問題視することにしました。ゲノム編集食品の問題点はオフターゲットという未知のリスクがあること、また国民のゲノム編集食品の安全性の認知度が低いことが挙げられます。



4．解決策

オフターゲットを防ぐ策がいくつかあるのです。ゲノム情報が解読されている生物をゲノム編集する場合には、ゲノムの中に、狙った部位の塩基配列と同じ配列や似た配列がないか事前に調べます。同じ配列が別の遺伝子にある場合は、その配列を用いたゲノム編集は断念します。似た配列がある場合には、ゲノム編集をした後にも調べて、似た配列のところでは切れていない、変異していない、ということを確認したうえで商用化に踏み切る(11)ことで問題が解決します。ですからオフターゲットを防ぎ、ゲノム編集食品の普及につながると考えられます。 また、国民のゲノム編集食品の安全性の認知度を高めるには国民のゲノム編集食品の関心を持たせ、安全性があることを知ってもらう必要があります。関心を持たせるにはゲノム編集の概要を政府がHPやテレビで発信し、国民に注目させること、またゲノム編集食品が食品売り場に売られたときには食品の商品名のすぐそばにゲノム編集食品はこれだけ安全性があるということを表示させることが安全性があることを知ってもらうきっかけになると考えられます。



5．まとめ

遺伝子操作食品はゲノム編集食品と遺伝子組み換え食品に分類されます。遺伝子組み換えのメリットはアレルギー物質を除いたり、安定に生産・供給ができることです。また、デメリットは健康被害がみられることです。それに比べてゲノム編集食品は短期間で品種改良ができ、人や環境に悪影響を及ぼすことが無いという点です。しかし、ゲノム編集食品は狙った遺伝子ではない別の遺伝子を切ってしまうオフターゲットという未知のリスクがあります。その問題を解決するためには、ゲノムの編集時に塩基配列の確認を重要視することでオフターゲット変異を防ぐこと、また政府がHPを作る、商品の説明欄に安全性を示すなどゲノム編集食品の安全性の認知度を高めることが必要であると考えられました。
